1. HTTP is a format to retrieve or send data to a server2. A URL is an address to an internet resource3. DNS converts URLs into IP addresses4. A query string are key-value pairs that add specifications to a request5. Get: retrieve data from a server. Post: send data to a server which changes a server.6. An HTTP request requests info from a server in HTTP format7. The response is what the server returns from a request8. Headers give additional info about a request or response: i.e. Host, Content-Type9. A URL is converted into an IP address using DNS and a request is made to that IP address and a response is sent with a status code. A DOM is created from the HTML and other resources are loaded via separate requests and responses for each resource.
